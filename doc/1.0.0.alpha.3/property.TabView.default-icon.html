AdwTabView:default-icon

property default-icon: Gio.Icon [ read, write ]

Description [src]

Default page icon.

If a page doesn’t provide its own icon via AdwTabPage:icon, a default icon may be used instead for contexts where having an icon is necessary.

AdwTabBar will use default icon for pinned tabs in case the page is not loading, doesn’t have an icon and an indicator. Default icon is never used for tabs that aren’t pinned.

By default, the adw-tab-icon-missing-symbolic icon is used.
