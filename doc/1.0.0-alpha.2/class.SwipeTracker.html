A swipe tracker used in AdwCarousel and AdwLeaflet.

The AdwSwipeTracker object can be used for implementing widgets with swipe gestures. It supports touch-based swipes, pointer dragging, and touchpad scrolling.

The widgets will probably want to expose the AdwSwipeTracker:enabled property. If they expect to use horizontal orientation, AdwSwipeTracker:reversed can be used for supporting RTL text direction.
