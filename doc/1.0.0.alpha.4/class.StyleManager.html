A class for managing application-wide styling.

AdwStyleManager provides a way to query and influence the application styles, such as whether to use dark or high contrast appearance.

It allows to set the color scheme via the AdwStyleManager:color-scheme property, and to query the current appearance, as well as whether a system-wide color scheme preference exists.
